# Учебный проект по интеграции программных модулей

Проект подготовлен для учебной практики по ПМ.02 "Осуществление интеграции программных модулей".

Цель проекта - показать базовую интеграцию модулей веб-приложения: разметки, стилей, модуля данных и клиентской логики. В рамках работы используется Git для фиксации изменений, ветвления, слияния, тегирования и имитации удаленного репозитория.

## Состав проекта

- `index.html` - главная страница учебного веб-приложения.
- `css/styles.css` - оформление интерфейса.
- `js/modules/tasks.js` - модуль с учебными задачами практики.
- `js/app.js` - клиентская логика вывода данных на страницу.
